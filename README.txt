Personal Expense Tracker - Python Mini Project

This project allows users to log daily expenses, view summaries, and track spending over time.

Features:
- Add expenses under various categories
- View summaries by category and date
- Edit or delete existing expenses
- View graphical summary (optional)
- Save/load data using file handling (JSON format)

How to Run:
1. Make sure Python is installed.
2. Install matplotlib: pip install matplotlib
3. Run the program: python expense_tracker.py

Files Included:
- expense_tracker.py: Main Python file
- expenses.json: Sample data file
- README.txt: Instructions
